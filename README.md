# 💱 Conversor de Moedas

Meu primeiro projeto desenvolvido com **HTML**, **CSS** e **JavaScript**, criado durante os estudos no DevClub.

O objetivo do projeto é converter valores em **Real Brasileiro (BRL)** para outras moedas, atualizando a interface de forma dinâmica conforme a moeda selecionada.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (ES6)

## ✨ Funcionalidades

* Conversão de Real Brasileiro para:

  * 🇺🇸 Dólar Americano (USD)
  * 🇪🇺 Euro (EUR)
  * 🇬🇧 Libra Esterlina (GBP)
* Atualização dinâmica da moeda selecionada
* Alteração da imagem da moeda
* Formatação automática dos valores utilizando `Intl.NumberFormat`
* Interface simples e intuitiva

## 📚 O que pratiquei neste projeto

* Manipulação do DOM
* Eventos com `addEventListener`
* Funções em JavaScript
* Estruturas condicionais (`if`)
* Seleção de elementos com `getElementById` e `querySelector`
* Formatação de moedas com `Intl.NumberFormat`
